# Pulverize
![Pulverize Logo](https://i.postimg.cc/FHSYvdsg/Untitled.png)

Pulverize, is a piece of code that makes hack client development for browser games way more easier.

This is made mainly for games such as:
- Bloxd.io
- Miniblox.io
- minefun.io
etc..

This code is open-sourced and is free to be copied or re-used by any developer. It is also extremely customizable and easy to change.
